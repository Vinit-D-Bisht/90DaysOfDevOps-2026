Process Management
ps aux        – Show all running processes
top           – Real-Time process view
htop          – Interactive process view
kill <PID>    – Kill process by its ID
kill -9 <PID> – Force kill process by its ID
pkill <name>  – Kill process by Name
jobs          – List background running jobs
fg %1         – Bring first job to foreground
bg %1         – Resume first job in background

File System
ls -l         – List files with details
ls -a         – Show hidden files
cd <dir>      – Change directory
pwd           – Show current present directory
mkdir <dir>   – Create directory
rm <file>     – Delete file
rm -r <dir>   – Delete directory recursively or non-empty file
cp <src> <dest> – Copy files from source to destination
mv <src> <dest> – Move or rename files
find /path -name <file> – Search file with name
du -sh <dir>  – Show directory size
df -h         – Show disk space usage

Networking
ping <host>   – Test connectivity
ip addr       – Show IP addresses
dig <domain>  – Query DNS records
curl <url> – Fetch content from url
